# 訪問看護スケジュール管理システム

スリーピース株式会社向け 訪問看護ステーションのスケジュール管理Webアプリケーション

## 機能一覧

### 📅 週間カレンダー
- スタッフ別・利用者別の切り替え表示
- 訪問予定のドラッグ＆ドロップ編集
- 週送り/戻し、今週ボタン
- 介護保険/医療保険のフィルタリング

### 🟢 空き状況表示
- **カレンダー常時表示**: 各セルに空きスタッフ数をヒートマップ色で表示（ON/OFF切替可能）
- **空き状況一覧パネル**: 時間帯別の空きスタッフ数をヒートマップで一覧表示
- **セルクリック詳細**: 任意の曜日×時間帯をクリックして全スタッフの空き/予定を確認
- **スタッフ別一覧**: 各スタッフの週間予定を一覧で確認

### 👤 利用者管理
- 利用者の新規追加（名前・住所・エリア・保険種別・サービスコード・定期訪問スケジュール）
- 利用者詳細表示（基本情報・保険情報・定期スケジュール・今週の訪問予定）
- 名前・住所・サービスコードで検索

### 🏥 保険・サービスコード対応
- 介護保険（訪看I1〜I5、予防訪看）
- 医療保険（基本療養費、管理療養費、加算等）
- カード上に保険種別バッジとサービスコードを表示

## セットアップ

```bash
npm install
npm run dev
```

ビルド: `npm run build`

## 技術スタック

- React 18 + Vite
- インラインCSS（ゼロ外部依存）

## ディレクトリ構成

```
src/
├── App.jsx                    # メインアプリ
├── main.jsx                   # エントリポイント
├── data.js                    # マスタデータ
├── styles.js                  # 共通スタイル
└── components/
    ├── ui.jsx                 # 共通UIコンポーネント
    ├── AvailabilityPanel.jsx  # 空き状況パネル
    ├── UserDetailPanel.jsx    # 利用者詳細
    └── AddUserModal.jsx       # 利用者追加
```

## カスタマイズ

- スタッフ変更: `src/data.js` の `STAFF`
- サービスコード追加: `src/data.js` の `SERVICE_CODES`
- エリア変更: `src/components/AddUserModal.jsx`
