# CLAUDE.md - 訪問看護スケジュール管理システム

## プロジェクト概要
スリーピース株式会社（訪問看護ステーション）のスケジュール管理Webアプリ。
スタッフ8名、利用者約80名の訪問スケジュールを週間カレンダーで管理する。

## 技術スタック
- React 18 + Vite
- インラインCSS（外部CSSライブラリなし）
- 外部依存: react, react-dom のみ

## ディレクトリ構成
```
src/
├── App.jsx                    # メインアプリ（カレンダー・サイドバー・編集モーダル）
├── main.jsx                   # エントリポイント
├── data.js                    # マスタデータ（スタッフ・サービスコード・利用者生成）
├── styles.js                  # 共通スタイル定数（lbl, sty, navBtn）
└── components/
    ├── ui.jsx                 # 共通UI（InsBadge, StBadge, Section, InfoRow）
    ├── AvailabilityPanel.jsx  # 空き状況パネル（ヒートマップ・スタッフ別一覧）
    ├── UserDetailPanel.jsx    # 利用者詳細モーダル
    └── AddUserModal.jsx       # 利用者追加モーダル
```

## コマンド
- `npm install` - 依存パッケージインストール
- `npm run dev` - 開発サーバー起動（localhost:5173）
- `npm run build` - 本番ビルド（dist/に出力）

## コーディング規約
- 日本語コメント・ラベルを使用
- コンポーネントは関数コンポーネント + Hooks
- スタイルはインラインCSS（style={{}}形式）
- 共通スタイルは styles.js から import
- 共通UIコンポーネントは components/ui.jsx に集約

## ドメイン知識
- 保険種別: 介護保険 / 医療保険の2種類
- サービスコード: 介護保険（1311〜1414系）、医療保険（C005〜C011系）
- エリア: 柏エリア、高塚エリア、松戸エリア
- 営業時間: 8:00〜17:00
- 曜日: 月〜日（土日も訪問あり）

## 注意事項
- 利用者データは現在メモリ内で管理（永続化未実装）
- サービスコードは実際の介護保険・医療保険のコード体系に準拠
- スタッフの職種: 看護師、理学療法士(PT)、作業療法士(OT)、言語聴覚士(ST)
